This folder contains source code files auto-generated by Aspose.CsClassTransformer tool.
Do not change them manually since your changes will be lost during next auto-generation
session. Change the corresponding templates instead and regenerate source files by
Aspose.CsClassTransformer using AutoGenConfig.xml configuration file located in the root
folder of the solution. Change AutoGenConfig.xml to generate new classes based on the 
existing templates if needed.

Visit https://auckland.dynabic.com/wiki/display/org/2.+Class+generation for additional information.
INCLUDE_FILE directive is used to exclude template code from C++ porting, because C++ porter 
doesn't support Compile Action: None property.
