1. # **<a name="_toc194568321"></a>Overview/Purpose**
This process will provide guidelines on the detail of information required when authoring a Standard Operating Procedure (SOP) and/or Process Description (PD) document.

1. # <a name="_toc194568322"></a>**Scope**
This PD will cover the following document types: -

- Standard Operating Procedures
- Process Descriptions 
- Process
- Policy
