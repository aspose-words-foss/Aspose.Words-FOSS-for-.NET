  • PROCESSING THE BOLA VIA A TRANSACTION SET:
  • STEP ONE - CREDITING THE CUSTOMER BY BOLA:
    1. once all necessary authorisation has been granted
    2. access Browser Main Menu
    3. go to Transactions > Set > Input Set
    4. input the following: customer sort code and account number (no spaces) of the account to credit
    5. serial number - Enter '10'
    6. description - enter the required reference this will appear on the customer's statement. Write 'as advised' unless a specific reference is required
    7. amount - The GBP value e.g. 12.00
    8. from the drop down box select 'CR' (Credit)
    9. select >'Add item to transaction set'.
